# 社保业务前道客服Agent系统

一个用于处理社保增减员业务的自动化客服系统，支持Excel表格解析、字段校验、工单生成和错误跟踪。

## 项目结构

```
.
├── config/              # 配置文件
│   ├── __init__.py
│   └── settings.py      # 系统配置（字段规则、错误处理等）
├── models/              # 数据模型
│   ├── __init__.py
│   └── schema.py        # 增员/减员工单数据模型
├── modules/             # 核心模块
│   ├── __init__.py
│   ├── parser.py        # Excel表格解析
│   ├── validator.py     # 字段校验
│   ├── workorder.py     # 工单生成（按城市拆分）
│   └── state_manager.py # 状态管理和错误计数
├── examples/            # 示例文件
│   ├── create_examples.py
│   ├── 增员示例.xlsx
│   └── 减员示例.xlsx
├── output/              # 输出目录
│   └── workorders/      # 生成的工单文件
├── data/                # 数据存储
│   ├── state.json
│   └── customer_errors.json
├── logs/                # 日志文件
├── main.py              # 主程序入口
└── requirements.txt     # 依赖包
```

## 功能特性

### 1. Excel表格解析
- 支持增员和减员两种工单类型
- 自动识别工单类型（通过列名判断）
- 解析`.xlsx`和`.xls`格式

### 2. 字段校验
- 必填项检查
- 格式验证（身份证、电话、月份等）
- 选项验证（人员分类、户口性质等）
- 数值范围验证（社保基数、公积金基数）

### 3. 工单生成
- 按城市自动拆分工单
- 生成唯一工单号（WO+日期+UUID）
- 输出JSON格式工单文件

### 4. 错误跟踪
- 按客户统计错误次数
- 超过阈值自动转人工客服
- 错误计数自动重置（可配置天数）

## 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行Demo

```bash
python main.py --demo
```

### 处理指定文件

```bash
python main.py --file <Excel文件路径>
```

示例：
```bash
python main.py -f examples/增员示例.xlsx
```

## 配置说明

在`config/settings.py`中可以配置：

- **错误处理**：最大错误次数、重置天数
- **工单生成**：城市列表、拆分规则
- **验证规则**：字段格式、选项、数值范围
- **存储路径**：日志、工单、数据目录
